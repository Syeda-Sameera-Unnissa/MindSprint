# 🚀 MindSprint

MindSprint is a productivity web app designed to help users manage tasks, stay focused, and build consistency.

🔗 **Live App:** https://mind-sprint-sandy.vercel.app/

---

## ✨ Features

- 📝 Task & Todo Management  
- 🎯 Focus Mode  
- 📊 Dashboard & Insights  
- ⭐ Feedback System  
- 👤 User Authentication (Supabase)  
- 🌙 Dark Mode Support  
- 🤖 AI Engine Integration  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, Tailwind CSS  
- **Backend:** Supabase (Authentication + Database)  
- **Deployment:** Vercel  

---

## 🔐 Authentication

- Secure login & signup using Supabase  
- Protected routes with session-based access  
- Automatic redirection for unauthorized users  

---

## 🚀 Getting Started (Local Setup)

### 1. Clone the repository

    git clone https://github.com/Syeda-Sameera-Unnissa/MindSprint.git
    cd MindSprint

### 2. Install dependencies

    npm install

### 3. Run the app

    npm run dev

---

## 🔑 Environment Variables

Create a `.env.local` file in the root folder and add:

    NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
